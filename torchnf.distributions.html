class torchnf.distributions.Prior(batch_size: int = 1)

Bases: object

Base class for prior distributions.

The two essential requirements for a prior distribution are (a) that it can be (efficiently) sampled from, and (b) that the log- probability of a sample can be computed up to an unimportant normalisation. Hence, derived classes should override sample() and log_prob().

Objects derived from this class can also be iterated over:

prior = Prior(...)
prior = iter(prior)  # constructs a generator
sample, log_prob = next(prior)

This behaviour is similar to that of an torch.utils.data.IterableDataset, which means they can be used in place of a DataLoader in PyTorch Lightning.

Parameters

batch_size – The default batch size, i.e. number of independent and identically distributed data points to draw when sampling. Used where prior is called as in next(iter(self)).
